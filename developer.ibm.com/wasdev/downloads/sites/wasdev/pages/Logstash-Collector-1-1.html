Logstash Collector 1.1

Logstash collector gathers data from various sources and forwards the data to a Logstash server using Lumberjack protocol.

Command Line Install

To install the feature from the command line, type:
bin/installUtility install logstashCollector-1.1

Config Instructions

To use the feature at runtime add the following to your server.xml file
<featureManager>
  <feature>logstashCollector-1.1</feature>
</featureManager>
