Scaling Member for Node.js

The Scaling Member for Node.js allows a Node.js collective member to be managed by a Liberty Scaling Controller server. The Scaling Controller directs the Scaling Member for Node.js to start and stop based on the scaling policy. The Scaling Member for Node.js is a plugin module for the apiconnect-collective-member npm package. The Scaling Member for Node.js plugin provides the equivalent capability to the scalingMember-1.0 feature for the Liberty for the Node.js collective member.

Prerequisites

  • Node.js runtime is installed and properly configured. See Installing Node.js via package manager for more information.
  • apiconnect-collective-member npm package is installed. See Installing the collective member host in Installing API Connect Collective for more information.

Installation

Install the plugin from the command line by doing the following:
  • Download wlpn-cli-scaling-member-1.0.1.tgz using the Download button at the top of this page
  • From the command line, run: wlpn plugins:install wlpn-cli-scaling-member-1.0.1.tgz

Additional information

  • For the complete feature documentation, see the Installing scaling members for Node.js servers description in IBM Knowledge Center.
  • For more detailed information about the Scaling Controller, see the Setting up auto scaling for Liberty collectives description in IBM Knowledge Center.
