Liberty Asset Repository Service Client

LARS is an open source repository service for Liberty features. For more information on LARS and the management client, see https://github.com/WASdev/tool.lars/.

Installing the LARS client

The LARS client is a java application. It is distributed here as a binary zip, or you can build it from the LARS project source. To get started with the zip, download, unzip and ensure you have Java 7 on your path. Run:
bin/larsClient help
for usage information, or see the documentation in the github project.
