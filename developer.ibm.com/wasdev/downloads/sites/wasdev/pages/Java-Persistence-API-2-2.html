Java Persistence API 2.2

This feature enables support for applications that use application-managed and container-managed JPA. This feature includes Java Persistence API 2.2 specification interfaces and container-managed JPA integration. EclipseLink is also included as the default JPA provider implementation.

Command Line Install

To install the feature from the command line, type:
bin/installUtility install jpa-2.2
If you are installing into 8.5.5.5 or earlier and the feature supports that version, use the featureManager command, for example:
bin/featureManager install jpa-2.2 --when-file-exists=ignore

Config Instructions

To use the feature at runtime add the following to your server.xml file
<featureManager>
  <feature>jpa-2.2</feature>
</featureManager>
