IBM Monitoring and Diagnostics Tools - Garbage Collection and Memory Visualizer

The IBM Monitoring and Diagnostic Tools - Garbage Collection and Memory Visualizer (GCMV) is a tool which allows you to visualize and analyse the memory usage and garbage collection activity of your Java or JavaScript application.

Installation

Try GCMV in Eclipse by doing the following:
  1. If you don't already have Eclipse, install Eclipse Luna for Java EE Developers ( 4.4 )
  2. Select the Install button below, drop it onto the Eclipse toolbar, then follow the prompts.
