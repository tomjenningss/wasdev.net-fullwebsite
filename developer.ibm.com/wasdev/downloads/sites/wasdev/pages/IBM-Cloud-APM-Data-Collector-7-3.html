IBM Cloud APM Data Collector 7.3

This extension pack contains a data collector, which can be deployed for monitoring your Liberty servers with IBM Cloud Application Performance Management. This data collector is applicable to the Liberty server running on Linux for System x only

Note:This extension pack works with Cloud APM (SaaS) only. If you have an existing Cloud APM on-premises installation, do NOT use this extension pack to deploy the Liberty data collector. Instead, download the data collector package from your existing Cloud APM installation and follow the instructions on Cloud APM Knowledge Center to configure the data collector.


Prerequisites

  • You must have an Application Performance Management subscription. If you do not have one yet, go to the IBM Marketplace and request a free trial of Application Performance Management.

  • The monitor-1.0 feature is required by the data collector. If your Liberty system has access to the Liberty feature repository, the extension pack can automatically download and install this feature.

Installation

  1. Go to the root of your Liberty installation directory. For example, /opt/ibm/wlp.

  2. Create a file named apmdc.properties under the wlp/etc/extensions directory with the following content in this file:

com.ibm.websphere.productId=com.ibm.apm.dc

com.ibm.websphere.productInstall=apmdc/

  1. From the wlp/bin directory, install this extension pack with the following command.

./installUtility install apmDataCollector-7.3 --to=apmdc


Configuration

Use the config_liberty_dc.sh script in the < liberty_home>/../apmdc/7.3.0.14.04/bin directory to configure the data collector by responding to prompts.

Note: If you encounter curl: (35) SSL connect error during the configuration process, update the curl of your system to the latest version and try again.

Additional information

For complete documentation of this extension pack, see APM Developer Center.

For information about how to download the Liberty features with installUtility, see Downloading assets using the installUtility command in the WebSphere Application Server Network Deployment Knowledge Center.

For information about Cloud APM dashboards and advanced configuration of Liberty data collector, see the Cloud APM Knowledge Center.
