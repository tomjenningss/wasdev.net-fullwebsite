Java EE 7 Full Platform Bundle

This bundle provides a convenient combination of Liberty features that support the Java EE 7 Full Platform.

Command Line Install

To install the feature from the command line, type:
bin/installUtility install javaee7Bundle
If you are installing into 8.5.5.5 or earlier and the feature supports that version, use the featureManager command, for example:
bin/featureManager install javaee7Bundle --when-file-exists=ignore
