IBM Monitoring and Diagnostics Tools - Interactive Diagnostics Data Explorer (IDDE)

Interactive Diagnostics Data Explorer (IDDE) is a GUI-based dump viewer for exploring and examining dump files that are produced by the IBM Java Virtual Machine and the IBM SDK for Node.js.

Installation

Try IDDE in Eclipse by doing the following:
  1. If you don't already have Eclipse, install Eclipse Luna for Java EE Developers ( 4.4 )
  2. Select the Install button below, drop it onto the Eclipse toolbar, then follow the prompts.
