IBM CICS remote development feature for Java

This feature provides a JCA resource adapter for use in Liberty to call a program in CICS Transaction Server (CICS). The resource adapter connects to CICS using TCP/IP. The feature is for development use only.

Installation

To install the feature from the command line, type:

bin/installUtility install jcaRemoteEci-1.0

Config instructions

To use the feature add the following to your server.xml file. The hostname and port should be set to target the TCP/IP service (TCPIPSERVICE) resource installed in CICS.

<featureManager>
  <feature>usr:jcaRemoteEci-1.0</feature>
</featureManager>

<connectionFactory id="defaultCICSConnectionFactory"
  jndiName="eis/defaultCICSConnectionFactory">
    <properties.com.ibm.cics.wlp.jca.remote.eci
      serverName="tcp://hostname:port"/>
</connectionFactory>
