WebSphere eXtreme Scale for Developers Liberty Profile

WebSphere eXtreme Scale V8.6 for Developers - Liberty Profile processes massive volumes of transactions with extreme efficiency and linear scalability. WebSphere eXtreme Scale operates as an in-memory data grid that dynamically caches, partitions, replicates, and manages application data and business logic across multiple servers.

WebSphere eXtreme Scale V8.6 for Developers - Liberty Profile is a no charge distribution of WebSphere eXtreme Scale, ideal for projects that don't warrant the expense of a priced and supported runtime. It allows developers to test their applications and data grids on the WebSphere Application Server Liberty Profile.
