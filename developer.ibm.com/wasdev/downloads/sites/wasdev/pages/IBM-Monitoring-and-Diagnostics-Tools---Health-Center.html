IBM Monitoring and Diagnostics Tools - Health Center

Health Center is a quick to deploy, small impact live monitoring tool running on either IBM Java Virtual machines or IBM SDK for Node.js. By gathering information on a wide range of areas such as method profiling, Garbage collection and memory usage, Health Center provides recommendations and analysis to help you improve the performance and efficiency of your application.

Installation

Try Health Center in Eclipse by doing the following:
  1. If you don't already have Eclipse, install Eclipse Luna for Java EE Developers ( 4.4 )
  2. Select the Install button below, drop it onto the Eclipse toolbar, then follow the prompts.
