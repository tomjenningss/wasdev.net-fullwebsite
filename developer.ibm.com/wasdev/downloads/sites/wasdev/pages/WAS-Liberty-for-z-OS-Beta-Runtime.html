WAS Liberty for z/OS Beta Runtime

The lightweight WebSphere Liberty for z/OS is production-ready and designed for developers. It is Java EE 7 certified.

Installation

Complete the following steps to install the WebSphere Liberty:

  1. Download WebSphere Liberty for z/OS Beta by clicking Download.
  2. Run the following command: cat wlp-beta-zos-2019.11.0.0.pax | pax -r -ppx
