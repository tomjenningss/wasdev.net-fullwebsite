]]> By: Eric https://developer.ibm.com/wasdev/docs/sample_ejbwebservicessample/#comment-11594 Sun, 08 Jun 2014 13:35:49 +0000 http://ibmdw.net/wasdev/sample_ejbwebservicessample/#comment-11594 In step 9, please notice the “-cp ＜path_to_compiled_SEI_class＞;＜wlp.install.dir＞\dev\api\spec\com.ibm.ws.javaee.ejb.3.1_1.0.0.jar” is used to configure classpath to all needed classes when running the command. Please note the semicolon “;” is used as delimiter between two paths.

But in your command “-cp ;\dev\api\spec\com.ibm.ws.javaee.ejb.3.1_1.0.0.jar”. There is only one path to the jar file, and there is no ＜path_to_compiled_SEI_class＞ before semicolon. If your compiled classes are placed in the current folder, you should write it as “-cp .\;\dev\api\spec\com.ibm.ws.javaee.ejb.3.1_1.0.0.jar”.
