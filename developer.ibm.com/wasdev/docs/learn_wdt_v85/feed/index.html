Server tools does not support JDK version higher than 8.

If the you really want to use JDK 11 to start Eclipse, the only workaround I can think of is to set the “Installed JRE” in Eclipse to use JDK8.

Window -> Preference -> Java -> Installed JREs ->add JDK8 and select the checkbox, then click “Apply”

I don’t think there’s any workaround other than that.
