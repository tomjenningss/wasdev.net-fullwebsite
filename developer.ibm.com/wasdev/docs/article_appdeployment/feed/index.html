]]> By: David Currie https://developer.ibm.com/wasdev/docs/article_appdeployment/#comment-147379 Wed, 14 Sep 2016 11:25:09 +0000 http://ibmdw.net/wasdev/article_appdeployment/#comment-147379 Hi Kevin – absolutely. In a Liberty collective the collective controllers provide the same function as a central point of administration as the deployment manager in a traditional cell. For example, a controller can act as a JMX proxy for any server in the collective. There are some significant differences though: there can be multiple controllers for starters and the configuration is owned by the individual servers rather than by the controller.
